Expense Tracker — это коллекция из семи независимых реализаций трекера личных расходов, каждая из которых работает на своём языке программирования. Проект создан для демонстрации кроссплатформенного подхода и предоставления удобного инструмента для учёта финансов с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
📝 Добавление расходов с категорией, суммой, датой и описанием

📂 Управление категориями (создание, редактирование, удаление)

📅 Фильтрация по дате, категории, диапазону сумм

📊 Статистика:

Общая сумма за период

Траты по категориям (с процентами)

Среднедневные/среднемесячные расходы

💾 Сохранение и загрузка из JSON/CSV

🔄 Экспорт отчётов в текстовый формат и CSV

📈 Графики (в некоторых версиях – ASCII-графики в консоли или интеграция с библиотеками)

🌍 Поддержка валют (настраиваемый символ валюты)

🖥️ Интерфейсы:

Командная строка (CLI) с интерактивным меню

Графический интерфейс (GUI) в Python, Java и C#

Веб-интерфейс в PHP (и браузерный JavaScript)

🚀 Быстрый старт
Установка зависимостей
Python
bash
pip install tkinter  # (опционально для GUI)
JavaScript (Node.js)
bash
npm install commander
Go
bash
go get -u
Rust
bash
cargo add clap serde_json
Java
Не требует внешних библиотек (используется стандартный java и javax.swing).

C#
Не требует внешних библиотек (используется System.Windows.Forms и System.Text.Json).

PHP
Не требует внешних библиотек.

Использование (общий синтаксис)
CLI (все версии):

bash
expense_tracker --add --category Food --amount 12.50 --description "Lunch"
expense_tracker --list --category Food --from 2024-01-01 --to 2024-01-31
expense_tracker --stats
Интерактивный режим (CLI):
Запустите без аргументов – откроется меню.
